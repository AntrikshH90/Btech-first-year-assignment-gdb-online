## üöÄ How to Use

1. **Clone the repository:**
   ```bash
   git clone https://github.com/AntrikshH90/Btech-first-year-assignment-gdb-online.git
   ```

2. **Compile any program:**
   ```bash
   gcc program_name.c -o program_name
   ./program_name
   ```

3. **Or use GDB Online Compiler:**
   - Visit [GDB Online](https://www.onlinegdb.com/)
   - Copy the code from any `.c` file
   - Paste and run!
